# Rick-And-Morty
Stack:
- react
- react-router
- react-query
- chacra-ui

# Installation
Install dependencies(make sure you have nodejs and npm or yarn installed)

    // with npm
    npm install

    // with yarn
    yarn install

Run it

    // with npm
    npm start

    // with yarn
    yarn start

I have trouble with CRA and framer-motion, so I use craco (you can check commit e014c27)

# IMHO
React-query is the best way to store any asynchronous data. Also chacra-ui is a very good components libary.
